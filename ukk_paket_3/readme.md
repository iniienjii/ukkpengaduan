Alur Sistem

Siswa
1. Login ke sistem  
2. Mengisi laporan aspirasi  
3. Mengirim laporan  
4. Melihat status dan histori  
5. Menerima feedback dari admin  

---

Admin
1. Login ke sistem  
2. Melihat data aspirasi  
3. Memproses laporan  
4. Memberikan feedback  
5. Mengubah status (Diproses / Selesai)
